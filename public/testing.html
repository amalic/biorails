This is my check list of urls which should be working

Admin Subsystem

  • Catalogue: to manage the namespace for lookups
  • DataFormats: to control the format of entered data
  • ParameterType: to manage type definition for linkage to warehouse
  • ParameterRole: to govern usage
  • DataType: Number,Text etc
  • Role: to control access to function
  • User: to identify the user

Project Subsystem

  • Project: High level management domain for division of work
  • Folder: Project folder to separation of work
  • article: Textual block
  • asset: File
  • ref: Reference to a remote object

Ogranization Subsystem

  • Study: Logical block of work
  • Protocol: Organized data entry sheet
  • Parameters: Data Parameter used in the study
  • Queue: Service queue
  • QueueItem:Item to process

Execution Subsystem

  • Experiment: Collection of protocol run together
  • Task: a protocol run once
  • Request: Request to do stuff to stuff
  • Service: Request to submit stuff to a queue
